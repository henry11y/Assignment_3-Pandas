# Assignment_3-Pandas

## Purpose
The purpose of this project is to combine Petal and Sepal iris datasets into one DataFrame and performs statistical analysis using pandas.

## What the Program Does
- Loads 'Petal_Data.csv' and 'Sepal_Data.csv'
- Cleans colums names for consistency
- Merges both datasets into one DataFrame
- Calculates:
    - 6 pairwise correlations
    - Mean, median, and standard deviation grouped by species
    - Determones with species are most similar and least similar based on mean measurements
 
## Tools Used
- Python
- Pandas
